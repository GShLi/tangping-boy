# 安装 Kubernetes

## 通过 yum 安装

### 1. 准备国内 yum 源

```bash
cat <<EOF > /etc/yum.repos.d/kubernetes.repo
[kubernetes]
name=Kubernetes
baseurl=https://mirrors.aliyun.com/kubernetes/yum/repos/kubernetes-el7-x86_64/
enabled=1
gpgcheck=1
repo_gpgcheck=1
gpgkey=https://mirrors.aliyun.com/kubernetes/yum/doc/yum-key.gpg https://mirrors.aliyun.com/kubernetes/yum/doc/rpm-package-key.gpg
EOF
```

### 2. 安装

```bash
setenforce 0

yum install -y --nogpgcheck kubelet kubeadm kubectl
```

### 3. 启动

```bash
systemctl start kubelet

systemctl enable kubelet

swapoff -a
```

### 4. 配置初始化文件

```bash
kubeadm config print init-defaults > init.default.yaml
```

```bash
registry.aliyuncs.com/google_containers
```

### 5. 提前拉取镜像

查看镜像列表

```bash
kubeadm config images list
```

pull 镜像

```shell
images=(
    kube-apiserver:v1.25.3
    kube-controller-manager:v1.25.3
    kube-scheduler:v1.25.3
    kube-proxy:v1.25.3
    pause:3.8
    etcd:3.5.4-0
    coredns:v1.9.3
)

for imageName in ${images[@]} ; do
    docker pull registry.cn-hangzhou.aliyuncs.com/google_containers/$imageName
    docker tag registry.cn-hangzhou.aliyuncs.com/google_containers/$imageName k8s.gcr.io/$imageName
    docker rmi registry.cn-hangzhou.aliyuncs.com/google_containers/$imageName
done

kubeadm config images pull

docker pull registry.cn-hangzhou.aliyuncs.com/google_containers/coredns:v1.9.3
docker tag registry.cn-hangzhou.aliyuncs.com/google_containers/coredns:v1.9.3 k8s.gcr.io/coredns/coredns:v1.9.3

docker pull registry.cn-shanghai.aliyuncs.com/double-registry/double/coredns:v1.9.3
docker tag registry.cn-shanghai.aliyuncs.com/double-registry/double/coredns:v1.9.3 k8s.gcr.io/coredns/coredns:v1.9.3

registry.cn-shanghai.aliyuncs.com/double-registry/double
```

### 6. Master Node 初始化

```bash
kubeadm init
```

::: warning 初始化报错

```text
unknown service runtime.v1alpha2.RuntimeService
```

解决方案

```bash
rm -f /etc/containerd/config.toml
systemctl restart containerd
```

`docker ps -a` 查看 apiserver 对应的容器有没有起来

`docker logs <containerID>` 查看容器日志
