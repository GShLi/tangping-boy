# yum 源搭建及更换

参考文章：<https://blog.csdn.net/hello_xiaozhuang/article/details/80112380>

## 修改 yum 源

```bash
# 移除旧 yum 源
mv /etc/yum.repos.d/CentOS-Base.repo  /etc/yum.repos.d/CentOS-Base.repo.bak

# 下载阿里云的 yum 源
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

# 清除系统原来的 yum 缓存
yum clean all
# 生成 yum 缓存，即可使用 yum 安装 yum 中的数据
yum makecache
```

## 卸载软件

```bash
yum erase 软件包名

```
