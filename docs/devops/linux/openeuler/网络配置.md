# Openeuler 网络配置

## 处理网络问题：Network is unreachable

[参考文章]<https://docs.redhat.com/zh_hans/documentation/red_hat_enterprise_linux/7/html/networking_guide/sec-configuring_ip_networking_with_ifcg_files>

```bash
vim /etc/sysconfig/network-scripts/ifcfg-ens192
```

```text
TYPE=Ethernet
PROXY_METHOD=none
BROWSER_ONLY=no
BOOTPROTO=static
DEFROUTE=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=yes
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_FAILURE_FATAL=no
NAME=ens192
UUID=d2d0b552-caae-4072-af46-84b31aeb64a8
DEVICE=ens192
ONBOOT=yes
IPADDR=192.168.0.116
PREFIX=24
DNS1=192.168.0.1
DNS2=192.168.0.114
DNS3=192.168.1.1
```

OpenEuler 网络连接报错

```bash
[root@localhost network-scripts]# ping www.baidu.com
ping: connect: Network is unreachable
```

```bash
# 检查路由信息
ip route show
# 修改路由信息 ---ok 问题解决
sudo ip route replace default via 192.168.0.1 dev ens192
```
