## 关于本站

这里是 **Double** 的个人网站。我是一名 Java 开发者，目前在业务架构方向深耕。

AI 时代，知识的获取成本趋近于零。本站不追求记录"是什么"，而专注于记录**"我是怎么想的"**——技术决策背后的权衡、踩过的坑、读过的书引发的思考。

- **博客**：技术复盘、读书笔记、个人随笔
- **学习笔记**：Java 生态体系化知识，按需深入而非面面俱到
- **项目**：工作实践的设计文档，如 Titan ETL 数据平台

> 核心信念：**AI 能生成答案，但不能替你思考。**
