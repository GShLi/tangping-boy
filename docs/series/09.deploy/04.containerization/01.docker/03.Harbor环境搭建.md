# Harbor 2.10.0 搭建

[Harbor 官方安装文档](https://goharbor.io/docs/2.10.0/install-config/)

Harbar 版本：2.10.0

Docker Engine 版本：25.0.3

## 安装需求

[Harbor 安装需求](https://goharbor.io/docs/2.10.0/install-config/installation-prereqs/)

## 离线安装步骤

1. 下载离线安装包及 [ASC 校验文件](/docs/devops/linux/ASC 校验文件.md)

    [Harbor Release](https://github.com/goharbor/harbor/releases)

    ```bash
    wget -O /opt/install/harbor-offline-installer-v2.9.2.tgz https://github.com/goharbor/harbor/releases/download/v2.9.2/harbor-offline-installer-v2.9.2.tgz
    ```

2. 下载公钥
3. 解压安装包

    ```bash
    tar xzvf harbor-offline-installer-v2.9.2.tgz -C /opt/install/
    ```

4. 配置 Harbor YML 文件

    ```bash
    cd /opt/install/harbor
    cp harbor.yml.tmpl harbor.yml
    ```

5. 执行安装脚本

    ```bash
    cd /opt/install/harbor
    bash install.sh
    ```
