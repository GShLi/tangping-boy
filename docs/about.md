# 关于我

## 我是谁

你好，我是 **Double**，一名 Java 开发者，目前在业务架构方向深耕。

日常和 Spring 全家桶、MyBatis、Redis、MySQL 打交道，对分布式系统和架构设计保持着持续的好奇心。

## 技术栈

- **语言**：Java（主力），会一点 Python 和 JavaScript
- **框架**：Spring / Spring Boot / MyBatis
- **中间件**：Redis / MySQL / Docker
- **工具**：Git / GitLab CI/CD / Linux
- **关注方向**：业务架构、分布式系统、领域驱动设计

## 爱好

- **阅读**：技术书和非技术书都读。最近越来越觉得，一本好的哲学书对思维的训练不亚于一本技术经典
- **哲学**：从一个奇怪的角度帮助我理解计算机科学中的抽象——说到底，面向对象不也是一种"理念论"吗？
- **摄影**：按下快门的瞬间和写下一段优雅代码的瞬间，有某种相似的美感
- **徒步**：和 debug 一样，都是寻找出路的过程

## 关于本站

本站是个人思考、记录与成长的数字花园。

在 AI 能一键生成技术教程的时代，本站不追求记录"是什么"，而专注于记录**"我是怎么想的"**。每一篇文章背后，都是一次真实的实践、一个实际解决的问题、或一本让我有所触动的书。
