# yum 源搭建及更换

参考文章：<https://blog.csdn.net/hello_xiaozhuang/article/details/80112380>

## 修改 yum 源

```bash
# 移除旧 yum 源
mv /etc/yum.repos.d/CentOS-Base.repo  /etc/yum.repos.d/CentOS-Base.repo.bak

# 下载阿里云的 yum 源
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

# 清除系统原来的 yum 缓存
yum clean all
# 生成 yum 缓存，即可使用 yum 安装 yum 中的数据
yum makecache
```

## 查找

```bash
# 默认 list 只显示最新的，通过 --showduplicate 可查看所有的
yum list kubeadm --showduplicates | sort -r
```

## 卸载软件

```bash
yum erase 软件包名

```

## 下载全量依赖 rpm 包及离线安装

```bash
# 安装yum-utils
yum -y install yum-utils

# 下载 ansible 全量依赖包
repotrack ansible

# 离线安装
rpm -Uvh --force --nodeps *.rpm
```
