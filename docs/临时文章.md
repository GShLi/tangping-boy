# Java Socket

## 分类

Java socket 分为 普通 Socket 和 NioSocket

Buffer

属性列表：

- capacity: buffer 最大容量
- limit: 最大访问值，不能超过 capacity
- position: 当前操作元素的索引，不能超过 limit
- mark: 暂存 position，不能超过 position

恢复暂存的 position

```java
public Buffer reset() {
    int m = mark;
    if (m < 0)
        throw new InvalidMarkException();
    position = m;
    return this;
}
```

清空 buffer

```java
public Buffer clear() {
    position = 0;
    limit = capacity;
    mark = -1;
    return this;
}
```

将当前处理的索引值设置为 0，这样就可以从起始位置读取内容。

```java
public Buffer flip() {
    limit = position;
    position = 0;
    mark = -1;
    return this;
}
```

## Spirng 接口命名规范

xxxAware 表示对 XXX 可以感知的，当我们创建的某个类需要用到 spring 里面的东西时可以通过继承 xxxAware 相关的接口并实现相应的 setXXX 方法，spring 会自动注入。

例如：Spring 中的 EnvironmentAware 接口，其定义了 `void setEnvironment(Environment environment);` 方法，当我们的类实现了该接口并重写了 setEnvironment 方法时，Spring 会通过该方法将 `Environment` 传递给我们。

同样的 Spring 中还有类似 XXXCapable 的接口，标识具有提供 XXX 的能力，可以通过 getXXX 方法从实现该接口的类中获取某些东西。

例如：Spring 中的 EnvironmentCapable 接口，其定义了 `Environment getEnvironment();` 方法，因此可以从实现该接口的类中获取 `Environment` 对象。
