# Redis 单机数据库的实现

## 1. 数据库

Redis 服务器的信息保存在 `redisServer` 的数据结构中。

`server.h`:

```c
struct redisServer {
    /* General */
    pid_t pid;                  /* Main process pid. */
    redisDb *db;
    /* Configuration */
    int dbnum;                      /* Total number of configured DBs */
}
```

db 是一个指向数组的指针，该数据保存了 Redis 服务器所有的数据库信息。

dbnum 则表示当前 Redis 服务器有多少个数据库。
