# 安装 Solr

## Docker 安装

1. 创建 docker compose 文件

    ```yml
    version: '3.1'
    services:
    solr:
        image: solr:8.11.1
        restart: always
        container_name: solr
        environment:
        - TZ=Asia/Shanghai
        ports:
        - 8983:8983
        volumes:
        - ./solrdata:/opt/solrdata
    ```

2. 启动容器
   
   `docker compose up`

3. 检查容器状态

   `docker compose ps`

    可以看到如下内容:

    ```bash
    WARN[0000] /opt/solr/docker-compose.yml: `version` is obsolete 

    NAME      IMAGE         COMMAND                  SERVICE   CREATED         STATUS         PORTS
    solr      solr:8.11.1   "docker-entrypoint.s…"   solr      2 minutes ago   Up 2 minutes   0.0.0.0:8983->8983/tcp, :::8983->8983/tcp
    ```

## 创建 Solr 实例

```bash
# docker exec -ti <容器ID> solr create_core -c <实例名称>
docker exec -ti 5107ccdbedb2 solr create_core -c test_core
```

## 安装 IK 中文分词器

```xml
<fieldType name"text_ik" class="solr.TextField">
    <analyzer type="index">
        <tokenizer class="org.wltea.analyzer.lucene.IKTokenizerFactory" useSmark="false" conf="ik.conf"/>
        <filter class="solr.LowerCaseFilterFactory"/>
    </analyzer>
    <analyzer type="query">
        <tokenizer class="org.wltea.analyzer.lucene.IKTokenizerFactory" useSmark="true" conf="ik.conf"/>
        <filter class="solr.LowerCaseFilterFactory"/>
    </analyzer>
</fieldType>
```
