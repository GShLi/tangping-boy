# 分类维护 - 树形结构构建方法

## 目标结构

```yaml
Node:
    - NodeId
    - ParentNodeId
    - NodeName
    - ChildrenNode
```

## 方法一：Java Stream + 递归算法

```java
public List<CategoryVO> tree () {
  // 1. 查找所有的分类信息
  List<Category> allCategories = baseMapper.selectList (null);
  List<CategoryVO> categoryVOS = CategoryWrapper.build ().listVO (allCategories);
  // 2. 将分类信息组装成树形结构
  List<CategoryVO> rootCategories =
      categoryVOS.stream ()
          .filter (it -> it.getParentCid () == 0)
          .peek (it -> it.setChildren (findChildrenCategory (it, categoryVOS)))
          .collect (Collectors.toList ());
  return rootCategories;
}

// 递归算法
public List<CategoryVO> findChildrenCategory (CategoryVO root, List<CategoryVO> allCategory) {
  return allCategory.stream ()
      .filter (it -> Objects.equals (root.getCatId (), it.getParentCid ()))
      .peek (it -> it.setChildren (findChildrenCategory (it, allCategory)))
      .collect (Collectors.toList ());
}
```
