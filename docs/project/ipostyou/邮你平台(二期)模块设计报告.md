# “邮你”（I Post You）平台模块设计报告 （二期） 

## 二、模块详细功能清单

### 模块1：商家服务体系

| 功能点                  | 说明                                                                 |
|-------------------------|----------------------------------------------------------------------|
| 地域文化认证            | 需提交“特色证明”（如非遗证书/景区授权）                             |
| **三阶物流方案**        | 平邮（¥0附票）/挂号信（+¥3可追踪）/快递（+¥10次日达）               |
| 一键转寄功能            | 购买时选择：存库/寄好友/直发悬赏任务                                |
| 店铺运营工具            | 销量地图（查看客户分布）、限时折扣、专属邮票设计权                  |
| 文旅合作入口            | 对接地方文旅局资源，推送“城市代言明信片”活动                        |

### 模块5：信誉与数据体系

| 维度          | 用户指标                          | 商家指标                          |
|---------------|-----------------------------------|-----------------------------------|
| **核心指标**  | 寄出率/签收率/悬赏完成度         | 发货时效/描述相符度/纠纷率        |
| **可视化**    | 五角星评级（1-5⭐️）               | 金牌/银牌/铜牌店铺标识            |
| **奖惩**      | 高信誉优先匹配权                  | 金牌店铺首页推荐位               |
| **风控**      | 连续2次未寄出冻结账户7天          | 纠纷率＞5%扣保证金                |
